PackageName:	baggedLocalOutlierFactor
      Title:	Filter implementing the Bagged LOF outlier/anomaly detection algorithm.
Description:	A filter that applies the Bagged LOF (Local Outlier Factor) algorithm to 
compute an "outlier" score for each instance in the data. For more information, see: 
Aleksandar Lazarevic, Vipin Kumar (2005). Feature Bagging for Outlier Detection. 
Proceedings of the Eleventh ACM SIGKDD International Conference on Knowledge Discovery 
in Data Mining, pp. 157-166.
